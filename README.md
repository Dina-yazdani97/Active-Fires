🌍Active Fires Map\
(30DayMapChallenge)

This repository, I show you how to create a active fire map for Iran with terra data using R programming language.

🧰 Tools & Libraries This project uses the following R packages:

library(tidyverse)\
library(sf)\
library(classInt)\
library(extrafont)\
library(data.table)\
library(gganimate)\
library(gifski)\
library(transformr)\
library(rio)\
library(showtext)
